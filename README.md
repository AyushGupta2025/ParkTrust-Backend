\# ParkTrust: AI-Enforced Immutable Parking



\*\*Hack4Delhi 2025 Submission\*\*



\## Problem

Solves revenue leakage and overcrowding in MCD parking lots using "Chain of Custody" logs.



\## Tech Stack

\* \*\*Backend:\*\* Python (FastAPI)

\* \*\*Algorithm:\*\* Manhattan Distance (Least-Distance Routing)

\* \*\*Security:\*\* Immutable Ledger Hashing (Simulated for Prototype)



\## How to Run

1\. `pip install -r requirements.txt`

2\. `uvicorn main:app --reload`

3\. Open `http://127.0.0.1:8000/docs` to test the API.
